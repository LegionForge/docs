# Brand & Media

Brand assets, logos, and press materials for LegionForge live in the dedicated media repo.

[:fontawesome-brands-github: github.com/LegionForge/media](https://github.com/LegionForge/media){ .md-button .md-button--primary }

## What's in the media pack

- LegionForge wordmark in SVG + PNG (light and dark variants)
- Guardian wordmark
- Per-project icons
- Color palette (hex + Pantone)
- Typography reference
- Press kit (one-page project summary, founder bio, contact)

## Usage

LegionForge logos and wordmarks are licensed for use **describing LegionForge** — for example, in articles, presentations, integration documentation, or comparison material. They are not licensed for use **implying endorsement** of a product, sponsorship of an event, or partnership.

For specific usage questions or to request high-resolution assets not in the public repo, email <info@legionforge.org>.

## Color palette

| Token | Hex | Use |
|---|---|---|
| accent | `#00ff88` | Primary brand color (LegionForge green) |
| accent2 | `#00ccff` | Secondary accent (used sparingly) |
| danger | `#ff4444` | Security warnings, denied actions |
| bg | `#0d1117` | Dark background |
| surface | `#161b22` | Card / panel background |

## Wordmark

When using the wordmark in body text, the canonical capitalization is **LegionForge** (one word, both Ls capitalized). Don't write *legionforge*, *Legion Forge*, or *legion-forge*.

The product is **LegionForge**.
The organization is **LegionForge** (same).
The website is **legionforge.org** (lowercase, like all domains).
