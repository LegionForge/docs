# Contribute

LegionForge is open source and welcomes contributions across the ecosystem — framework, Guardian, tools, apps, and these docs.

## How you can help

| What | Where |
|---|---|
| Report a bug or request a feature | The relevant project repo's Issues tab |
| Fix a bug or implement a feature | A pull request to the relevant project |
| Improve these docs | A pull request to [LegionForge/docs](https://github.com/LegionForge/docs) — every page has an edit pencil in the top-right |
| Add a new project | Open an issue on [LegionForge/LegionForge](https://github.com/LegionForge/LegionForge) to discuss first |
| Security disclosure | Email [security@legionforge.org](mailto:security@legionforge.org) — please **do not** open a public issue for vulnerabilities |
